## 全局安装 electron

> $ npm i -g electron electron-builder

无法安装?
```sh
$ vim ~/.npmrc
```
```
ELECTRON_MIRROR="https://cdn.npm.taobao.org/dist/electron/"
```

- `ng e-win`设置`"baseHref": "./"`， `ng serve`设置`"baseHref": "/"`
